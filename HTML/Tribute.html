HTML PROJECTS: My Page

Stats

  • Knowledge required: HTML only
  • Difficulty: Beginner friendly.
  • Estimate project completion time: 2 hours

Skill focus of the project

Some of the HTML skills that you will practice while doing this project are:

  1. Adding different types of text to a web page. You will be practicing using text types like headings, lists, and links.
  2. Changing emphasis on text using bold and italics.
  3. Embedding an image on a web page and adding image captions.

Main components of tribute page

You must include the following parts in your completed project:

  1. You must include a title or heading on the tribute page.
  2. You should include an image on the tribute page.
  3. You included image must have a caption.
  4. You should include a timeline of the life of the tribute in the form of a list.

Where to publish your work

  • Codepen

    • Codepen is the easiest one to setup. You just need an account to begin with. Then create a new pen and copy paste your HTML code there.

  • Github

    • This one takes time to setup. However, you can use the Github.com web based editor to upload
    or copy paste your HTML.

    Then you need to go to the repository setting to enable the projects to be viewed on (your-username).github.io/(repository-name).
